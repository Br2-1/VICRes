
VIC-Res: Representation and operation of reservoirs in the Variable Infiltration Capacity model

-------------- Requirements ------------
VIC-Res 
The routing component of VIC-Res is developed in FORTRAN. It requires the G77 compiler. VIC-Res has been tested on Linux Ubuntu 16.04 (Xenial Xerus) operating systems.
